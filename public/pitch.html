🛡️ Tech Defense

Architecture & DevOps (Press Down)

Why Next.js Standalone on Cloud Run?

`output: 'standalone'` drops heavy node_modules. Final container is 87MB: ultra-low cold starts, scale-to-zero efficiency.

How is backend verified securely?

We avoid heavy service accounts using Firebase Identity REST API to validate client ID tokens on-the-fly (Zero-trust).

How to handle React 18 StrictMode Maps?

Built a Module-level Promise Singleton to prevent double-initialization crashes.

How are CI/CD secrets managed?

Client variables via build args. Server keys (GEMINI_API_KEY) in GCP Secret Manager, mounted dynamically at runtime.
